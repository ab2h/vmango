Vmango

Interface for virtual machines management.

The main goal of project is not to provide a hypervisor configuration tool, because that problem already solved by many configuration management systems like Ansible or Puppet, but provide a convenient way to manage virtual machines on existing hypervisors.

It uses openstack compatible image format, is simple to install and stores all information about machines directly inside the hypervisor. If you have a handful of servers with virtual machines and don't want to install complex systems like Openstack or oVirt, you definitely should try it.
